- crond 程序是 linux 里负责执行计划任务的程序
	- 使用 crontab 命令调用 crond 进程
	- crond 程序每隔一分钟去检查是否有计划任务要执行
- 在 /etc/crontab 里面可以查看创建计划任务的格式
